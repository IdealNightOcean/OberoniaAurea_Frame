# 金鸢尾兰框架 (OberoniaAurea_Frame)

## 概述

**金鸢尾兰框架** 是一款专为 **RimWorld（环世界）** 游戏设计的 C# 模组开发框架。该框架提供了一套完整的工具集和扩展系统，旨在简化模组开发流程，增强游戏的可扩展性，为模组开发者提供强大而灵活的基础设施。

---

## 核心特性

### 🎯 任务系统扩展

提供丰富的任务生成和管理工具，支持复杂任务链的构建：

| 功能模块 | 说明 |
|---------|------|
| **QuestNode 扩展** | 20+ 自定义任务节点，支持派系获取、物品生成、世界对象创建等 |
| **QuestPart 扩展** | 任务部分组件，支持难民交互等特殊任务逻辑 |
| **任务关联接口** | 统一的任务关联管理机制 |

**主要任务节点包括：**
- `QuestNode_GetFaction` / `QuestNode_GetMutiFactions` - 派系获取
- `QuestNode_GenerateWorldObjectWithMutiFactions` - 多派系世界对象生成
- `QuestNode_GenerateInteractiveWorldObject` - 可交互世界对象生成
- `QuestNode_RequestThingCount` - 物品数量请求
- `QuestNode_SeasonalRestriction` - 季节限制

**特色组件：**
- `TradeRequestComp` - 贸易请求组件
- `CategoryTradeRequestComp` - 分类贸易请求组件
- `SaleRequestComp` - 销售请求组件

### 🏪 SiteTrader - 大地图商店

独立的大地图商店系统，实现 `ITrader` 接口，可在世界地图上创建可交易的商店：

| 功能 | 方法 | 说明 |
|-----|------|------|
| **初始化** | `SiteTrader(TraderKindDef, WorldObject, Faction, RulePackDef, int)` | 创建商店实例 |
| **生成物品** | `GenerateThings()` / `GenerateThings(PlanetTile)` | 生成商店库存 |
| **刷新库存** | `RefreshThings()` / `RefreshThings(PlanetTile)` | 刷新商店物品 |
| **定时刷新** | `TickInterval(int)` | 按间隔自动刷新 |
| **物品计数** | `CountHeldOf(ThingDef, ThingDef)` | 获取持有物品数量 |
| **玩家购买** | `GiveSoldThingToPlayer(Thing, int, Pawn)` | 出售物品给玩家 |
| **玩家出售** | `GiveSoldThingToTrader(Thing, int, Pawn)` | 从玩家购买物品 |
| **销毁商店** | `Destory()` | 清理并销毁商店 |

**特性：**
- 支持关联世界对象 (`WorldObject`)
- 可配置刷新间隔
- 支持派系关联
- 自动生成商店名称
- 完整的存档支持 (`IExposable`)

#### IPawnFloatMenuOptionProvider - Pawn 浮动菜单选项提供接口

为 Pawn 提供自定义浮动菜单选项，配合 `CompPawnFloatMenuOptionHandler` 组件使用。

| 成员 | 说明 |
|------|------|
| `IsSelectedPawnValid(Pawn)` | 检查选中的 Pawn 是否有效 |
| `GetFloatMenuOption(Pawn)` | 获取浮动菜单选项 |

**配套组件：** `CompPawnFloatMenuOptionHandler`
- `RegisterProvider(IPawnFloatMenuOptionProvider)` - 注册菜单选项提供者
- `DeregisterProvider(IPawnFloatMenuOptionProvider)` - 注销菜单选项提供者

---

#### IPawnPreApplyDamage - Pawn 伤害预处理接口

在 Pawn 受到伤害前进行拦截处理，支持伤害吸收和修改。

| 成员 | 说明 |
|------|------|
| `Priority` | 处理优先级，数值越大越先执行 |
| `PawnPreApplyDamage(ref DamageInfo, out bool absorbed)` | 伤害预处理，可修改伤害信息或吸收伤害 |

**配套组件：** `CompPawnPreApplyDamageHandler`
- `RegisterDamageProcessor(IPawnPreApplyDamage)` - 注册伤害处理器（按优先级排序）
- `DeregisterDamageProcessor(IPawnPreApplyDamage)` - 注销伤害处理器

---

#### IPawnRetentionHolder - Pawn 保留持有者接口

标记接口，当 Pawn 的容器（ParentHolder）实现此接口时，Pawn 不会被游戏自动销毁。

**用途：** 用于固定远行队、世界对象等需要长期持有 Pawn 的场景，防止 Pawn 因离开地图而被销毁。
